# AJÒ Beach visual sources

AJÒ uses photographs of the actual Sardinian locations represented in the demo.
The repository stores optimized WebP derivatives for reliable PWA rendering. Each
in-app result links back to its original Wikimedia Commons file page. Each derivative
remains available under the license listed for its source image.
